架构
====
物理地址空间：可执行内存；虚拟空间：0xC0000000 + 以下数字
``0x0`` -- 配置
``0x1 ~ 0x3F`` -- 寄存器x1 -- x15
``0x40`` -- 复位向量
``0x44`` -- 异常向量
``0x48`` -- 中断向量
``0x4C`` ~ 0x1FF -- 内存（512）（``.text + .bss + .data + heap + stack``）
``0x200 ~ end`` -- 非法内存（512）

配置寄存器 （``0xC0000000``）
========
物理内存大小：2^[2:0] * 512字节
